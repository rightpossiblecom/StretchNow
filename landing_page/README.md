# StretchNow - Web Portal

StretchNow is a simple app that reminds you to take short stretch breaks throughout the day. This repository contains the landing page website providing information about the app, including features, how it works, FAQ, and legal policies.

## Purpose
The StretchNow landing page focuses on encouraging movement and breaking long periods of sitting without making any medical or health claims. It positions the app as a simple tool for gentle stretch reminders.

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
